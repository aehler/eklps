-= Eklipse Adviser Chrome plugin =-

Обозначения кнопок:
  • Кнопки Т, М, Б, С, З, П, III, II, I, AA это фильтры по классу скачек. При нажатии на фильтр будут показаны только выбранные скачки, если снять все фильтры классов - покажутся все доступные скачки вне зависимости от класса
  • fl, sc - фильтр по типу скачек
  • Ж, К - тут все понятно... Фильтр строгий, то есть если выбран любой или оба - отрезаются общие скачки
  • S, S+, S++. По умолчанию показыватся все скачки, в которые технически можно записать лошадь. Чтобы ограничить поиск по классу резвости можно выбрать один из этих фильтров (они взаимоисключающие). S выдаст строгое соответствие, S+ в пределах одной ступени шкалы резвости, S++ в пределах двух ступеней
